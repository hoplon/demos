# TodoFRP

An implementation of [TodoMVC][4] using [Hoplon][1].

## Demo

[View the demo here][3].

## Dependencies

- java 1.8+
- npm

## Setup

Run:

    ```bash
    $ npm install
    ```

## Usage

1. Start the auto-compiler. In a terminal:

    ```bash
    $ npx shadow-cljs watch app
    ```


2. Go to [http://localhost:8000][2] in your browser.

[1]: https://hoplon.io
[2]: http://localhost:8000
[3]: https://hoplon.github.io/demos/todoFRP/
