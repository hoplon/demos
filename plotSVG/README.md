# plotSVG

A [Hoplon][1] demo that plots some data using SVG.

![example plot][4]

[Demo][3]

## Dependencies

- java 1.8+
- npm

## Setup

Run:

    ```bash
    $ npm install
    ```

## Usage

1. Start the auto-compiler. In a terminal:

    ```bash
    $ npx shadow-cljs watch app
    ```


2. Go to [http://localhost:8000][2] in your browser.
